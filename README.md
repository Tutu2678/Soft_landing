# Installation Steps:

git clone https://github.bath.ac.uk/nj585/softlanding.git

Install Python3

Install Anaconda

pip install -r requirements.txt

## PS:- For Running DQN, DDQN & PRE
pip install git+https://github.com/openai/gym.git@0.15.4

## PS:- For running PPO
pip install gym==0.26.2
